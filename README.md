Bookstore E-commerce

A responsive bookstore landing page built to practice modern front-end development with HTML, CSS, and JavaScript. The project focuses on responsive layouts, reusable components, clean UI design, and interactive user experience.

🌐 Web Deployed on Vercel → (pegá acá el link cuando lo deployes)

✨ Features
Responsive layout for desktop, tablet, and mobile devices
Modern bookstore landing page
Hero slider
Book category sections
Search bar interface
Interactive cards and hover effects
Clean and reusable component structure
Modern UI using Flexbox and CSS Grid
🛠️ Tech Stack
HTML5
CSS3
JavaScript (ES6)
📌 Background

This project was built to strengthen my front-end fundamentals before moving into React. It focuses on writing semantic HTML, creating responsive layouts with Flexbox and CSS Grid, organizing reusable UI components, and improving overall CSS architecture and design skills.

It also helped me practice building larger interfaces while maintaining clean project organization and consistent styling.
